\doxysubsection{Detailed Description}
An entity in the game\textquotesingle{}s component entity system. 

The \mbox{\hyperlink{classEntity}{Entity}} represents a collection of attributes, called Components, that together are acted upon by \mbox{\hyperlink{classEntitySystem}{Entity\+System}} objects that form a complete game world.

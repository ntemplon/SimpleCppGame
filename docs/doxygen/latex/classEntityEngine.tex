Adds the provided entity to the engine. The call site, the engine, and any systems that choose to will share ownership of the entity through copies of the shared\+\_\+ptr. Adds a system to the engine. 

Adds the provided system to the engine. The call site and the engine will share ownership of the system through the shared\+\_\+ptr created by the copy constructor when calling this method. Systems will have \mbox{\hyperlink{classEntityEngine_a3eafb99f52dfe9b79995c533d51d314e}{update()}} called on them each frame in the same order they are added to the engine, so if processing order is important, care should be taken when organizing calls to \mbox{\hyperlink{classEntityEngine_a4b1a151835e6a31360c68e8e8852ced9}{add()}}.

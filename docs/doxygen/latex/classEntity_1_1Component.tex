\doxysubsection{Detailed Description}
A component of an entity in the game\textquotesingle{}s component entity system. 

Components represent an aspect of an \mbox{\hyperlink{classEntity}{Entity}}. They store state information, such as position and sprite data, as well as mark the \mbox{\hyperlink{classEntity}{Entity}} as belong to a certain class of object, like being the player or an AI opponent.

\mbox{\hyperlink{classEntity_1_1Component}{Component}} types are identified with their ID, which is passed to the constructor. To avoid odd bugs and behavior, all instances of the same \mbox{\hyperlink{classEntity_1_1Component}{Component}} subclass should have the same ID, which is usually declared as a static member of the class or a preprocessor constant in the header file.
